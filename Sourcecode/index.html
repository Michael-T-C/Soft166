How to use the Password Checker

Type any password into the text box and the strength bar will increase as you type

This is a service generated for the purpose of educating people about password strengths and what constitutes a strong password

Cyber security is becoming increasingly prevalent in todays world, with passwords and biometrics being more important as data becomes more valuable

This site does not store/log any passwords entered
